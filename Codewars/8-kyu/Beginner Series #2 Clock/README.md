### Objective

- Write a Function that, given "H" for Hours, "M" for Minutes and "S" for Seconds, returns the sum of all in Milliseconds.
  
### Constraints

- H must be between 0 and 23
- M must be between 0 and 59
- S must be between 0 and 59

- None.
